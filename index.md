# 職務経歴書・スキルシート

## スキルサマリー

### プログラミング言語
- **PHP**: Laravel等のフレームワークを使用したWeb開発経験、パフォーマンス改善
- **Ruby**: Ruby on Rails、Webクローリング開発経験
- **JavaScript/TypeScript**: フロントエンド開発、React、Next.js、Vue.js使用経験
- **HTML/CSS/SCSS**: レスポンシブデザイン、モダンなUI実装
- **Dart**: Flutter使用によるモバイルアプリ開発経験
- **Node.js**: サーバーサイドJavaScript開発経験
- **Perl**: レガシーシステムの保守・改修経験

### データベース
- **MySQL/MariaDB**: 設計、チューニング、運用経験、パフォーマンス改善
- **PostgreSQL**: 基本的な操作、クエリ作成、Supabase経由での利用
- **Vercel**: サーバーレスでのDB運用経験

### インフラ・ツール
- **OS**: CentOS, Ubuntu, Amazon Linux, Linux
- **クラウド**: AWS (EC2, S3, ALB, Lambda, RDS), Google Cloud, Vercel
- **バージョン管理**: Git
- **CMS**: Movable Type
- **テストツール**: playwright (E2Eテスト)
- **負荷試験**: Apache Bench

### フレームワーク・ライブラリ
- **バックエンド**: Laravel, Ruby on Rails
- **フロントエンド**: React, Next.js, Vue.js, jQuery
- **モバイル**: Flutter
- **BaaS**: Supabase

## 強み・特徴

- 多様な技術スタックでの開発経験（Web、モバイル、AI連携）
- レガシーシステムのリニューアル・移行経験
- B2B、B2C両方のWebサービス開発経験
- フロントエンド・バックエンドの両方に対応可能
- パフォーマンス改善、負荷試験の実施経験
- リーダーとしてのチームマネジメント経験
- 外部システムとの連携・API設計・実装経験
- テスト自動化・CI/CD構築経験
